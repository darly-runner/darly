----
#### 1. 로그인 페이지 (소셜로그인)

<img src = "/uploads/22ac8a12648243c44e1e8958cb1a0d19/login.jpg" width="250px" >
<img src = "/uploads/adf5f2b15439b5fc508ed17e464bc9d4/login2.jpg" width="250px" />

- 로그인은 소셜로그인으로 진행됩니다 (OAuth)
- 구글로그인, 카카오 로그인 두가지로 가능합니다

-------

#### 2. 혼자 달리기 페이지

<img src = "/uploads/8bb4b0df81b937ce2b1fde1658f016f3/main1.jpg" width="250px" >
<img src = "/uploads/a930f5061fa310db31777be47232881c/main2.jpg" width="250px" />

<br/>

- 목표를 설정한 뒤 기록을 측정할 수 있습니다
- 기록측정은 위치정보를 바탕으로 기록됩니다
- 이동거리와 시간에 한계를 걸어두어 GPS 조작에 대비 하였습니다

<br/>

<img src = "/uploads/6c6c4b9386a4a3884825bec9ceb89721/run1.jpg" width="250px" >
<img src = "/uploads/364e20240c9636a82b277bf70204761a/run2.jpg" width="250px" />

- 달리기를 하는도중에 이동거리, 페이스, 속력 등을 확인할 수 있습니다
- 달리기가 끝난뒤에 조금더 상세한 정보를 확인할 수 있으며 저장할 수 있습니다

--------

#### 3. 크루 페이지

<img src = "/uploads/8c8b98f49e487ba5619822d8a5eadfb9/crew1.jpg" width="250px" >

- 크루원들끼리 경쟁방을 만들어서 함께 기록을 공유하며 달릴 수 있습니다
